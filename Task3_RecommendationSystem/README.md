## Simple Recommendation System

This project is a beginner-friendly **movie recommendation system** developed using **Python** as part of the **CodSoft AI Internship (Task 3)**.

### Features:
- Suggests movies based on user-preferred genre
- Handles invalid inputs gracefully
- Simple, beginner-friendly logic using Python dictionary

### Tools Used:
- Python
- Jupyter Notebook

### Project Files:
- `recommendation.ipynb` – Implementation of the recommendation system
- `output.png` – Screenshot of program output
- `README.md` – Project documentation

### How to Run:
1. Open `recommendation.ipynb` in Jupyter Notebook
2. Run all cells
3. Enter your favorite genre and see the recommended movies

### Learning Outcome:
- Basics of Python dictionaries and loops
- Handling user input and conditional statements
- Logical structuring of simple recommendation systems
